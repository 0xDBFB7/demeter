About Demeter

Demeter is a comprehensive system for processing and analyzing X-ray Absorption Spectroscopy data.

Demeter is:
  • currently at version 0.9.18.2
  • available for linux, Windows, and Macintosh
  • a set of perl modules and related files
  • a programming tool -- it is the thing from which applications are built
  • free software, freely available from a git server
  • actively developed and maintained
  • in use by its author and users for real data analysis problems
  • a front end to Feff and Ifeffit or Larch
  • the code base for Athena and Artemis
  • named for the Greek goddess of the harvest

Windows Installer & Updater
  1. Download the Installer package and double-click to install the base Demeter system (version 0.9.13)
  2. Download the Updater package and double-click to install the cumulative updates to version 0.9.18.2
  3. Download the Documentation package and double-click to update the Athena and Artemis Users' Guides

It's that simple! Follow this link for complete instructions of how to use the Windows Installer and Updater.

Macintosh Installation
  1. Carefully follow the steps to install Macports at http://www.macports.org/install.php.
  2. Open a new Terminal window and type the following: 
     sudo port install xorg-server demeter
    It will take a while depending on your computer and network speed.
  3. When it is done, launch Athena by typing the following in Terminal:
     athena
    Note: Make sure you are in a directory you have write permission to (like your home directory) because hidden files are created at launch time.

Please note that the Mac version is a work in progress. Some features may not be available. Please report problems to the Ifeffit mailing list.
