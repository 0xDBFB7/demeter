About Demeter

Demeter is a comprehensive system for processing and analyzing X-ray Absorption Spectroscopy data.

Demeter is:

  • currently at version 0.9.9
  • a set of perl modules and related files
  • a programming tool but not an application -- it is the thing from which applications are built
  • free software, freely available from a git server
  • under heavy development
  • in use by its author for real data analysis problems
  • a front end to Feff and Ifeffit
  • the code base that replaces Athena and Artemis
  • named for the Greek goddess of the harvest

Documentation

Programming documentation
This is the documentation embedded into each module which is part of the Demeter distribution. The PODs have been nicely formatted for html and organized into a hierarchical structure that looks like a CPAN project page.

Demeter Programming Guide
The DPG is a how-to guide written at a higher level than the programming documentation. It includes descriptions of how to use the main components of Demeter as well as several complete examples.

Athena Users' Guide
This is the manual for the older version of Athena. It has not yet been updated for the new, Demeter-based version. The new version is, however, sufficiently similar to the old that this is still useful for understanding how to use Athena effectively.

Artemis Users' Guide
This is the users' guide for the new version of Artemis. While it is unfinished, much of the text and many of the screenshots are in place. Keep checking back for updates.

Artemis instruction videos
These are screencast videos demonstrating the use of the Artemis program for EXAFS data analysis.

Lecture videos
In November 2011, I presented an XAS training course at the Diamond Light Source. All my lectures and software demonstrations were recorded along with the computer desktop. The Diamond technical staff did a great job assembling all of this material in streaming video. (Linux users: Click here for a hint about using Microsoft Media Server (mms:) links with Firefox.)

XAS Education
My collection of presentations that I give during XAS training courses. You can also find all the data used in the presentations and in the various videos linked above. PDF files of presentations and a zip file containing all the example data can be found at the XAS Education downloads page
