About Demeter

Demeter is a comprehensive system for processing and analyzing X-ray Absorption Spectroscopy data.

Demeter is:
  • currently at version 0.9.15
  • a set of perl modules and related files
  • a programming tool -- it is the thing from which applications are built
  • free software, freely available from a git server
  • actively developed and maintained
  • in use by its author and users for real data analysis problems
  • a front end to Feff and Ifeffit (and soon Larch!)
  • the code base for Athena and Artemis
  • named for the Greek goddess of the harvest

Windows Installer & Updater

Note: Release 0.9.13.1 for Windows corrects a severe problem in Artemis that was introduced in 0.9.11. You should update immediately!

  1. Download the Installer package and double-click to base Demeter system.
  2. Download the Updater package and double-click to install the cumulative updates.
  3. Download the Documentation package and double-click to install the Athena and Artemis Users' Guides.

It's that simple! Follow this link for complete instructions of how to use the Windows Installer and Updater.

Documentation

Athena Users' Guide: This is the users' guide for the current version of Athena

Artemis Users' Guide: This is the users' guide for the current version of Artemis.

Artemis instruction videos: These are screencast videos demonstrating the use of the Artemis program for EXAFS data analysis.

Lecture videos: In November 2011, I presented an XAS training course at the Diamond Light Source. All my lectures and software demonstrations were recorded along with the computer desktop. The Diamond technical staff did a great job assembling all of this material in streaming video. (Linux users: Click here for a hint about using Microsoft Media Server (mms:) links with Firefox.)

XAS Education: My collection of presentations that I give during XAS training courses. You can also find all the data used in the presentations and in the various videos linked above. PDF presentations are shown at my my Speaker Deck page.

Programming documentation: This is the documentation embedded into each module which is part of the Demeter distribution. The PODs have been nicely formatted for html and organized into a structure that resembles a CPAN project page.

Demeter Programming Guide: The DPG is a how-to guide written at a higher level than the programming documentation. It includes descriptions of how to use the main components of Demeter as well as several complete examples.

Demeter Changes list: The changes file for Demeter, Athena, Artemis, and Hephaestus. This version might be slightly more up-to-date.

Demeter To Do list: Bruce's "To Do" list for Demeter, Athena, Artemis, and Hephaestus. This version might be slightly more up-to-date.
