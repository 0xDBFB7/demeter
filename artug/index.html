EXAFS Data Analysis using Feff and Ifeffit

This document explains the operation of ARTEMIS, a program for analysis of EXAFS data using FEFF and IFEFFIT. It is built using Demeter.

DEMETER is:

  • a set of perl modules and related files

  • a programming tool but not an application – it is the thing from which applications are built

  • free software, freely available from a git server

  • actively developed and maintained

  • in use by its author for real data analysis problems

  • a front end to Feff and Ifeffit (and soon Larch!)

  • the code base for Athena and Artemis

  • named for the Greek goddess of the harvest

ARTEMIS is:

  • a graphical front-end for FEFF and IFEFFIT built using DEMETER

  • a tool which makes easy analysis problems easy and hard analysis problem possible

  • in use by hundreds of scientists world-wide
