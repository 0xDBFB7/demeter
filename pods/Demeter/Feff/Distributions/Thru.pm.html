NAME

Demeter::Feff::Distributions::Thru - Histograms for MS paths through the absorber

VERSION

This documentation refers to Demeter version 0.9.24.

SYNOPSIS

DESCRIPTION

This provides methods for generating two-dimensional histograms in path length and forward scattering angle for nearly collinear arrangements of three atoms, like so:

   Scatterer --- Absorber --- Scatterer

Given radial ranges for the near neighbor scatterer and its ipot, compute a two-dimensional histogram in half-path-length and scattering angle through the absorber.

ATTRIBUTES

file (string)

The path to and name of the HISTORY file. Setting this will trigger reading of the file and construction of a histogram using the values of the other attributes.

nsteps (integer)

When the HISTORY file is first read, it will be parsed to obtain the number of time steps contained in the file. This number will be stored in this attribute.

rmin and rmax (numbers)

The lower and upper bounds of the radial distribution function to extract from the cluster.

rbin (number)

The width of the histogram bin to be extracted from the RDF.

betabin (number)

The forward scattering angular range through the absorber of the histogram bin to be extracted from the RDF.

sp (number)

This is set to the Demeter::ScatteringPath object used to construct the bins of the histogram. A good choice would be the similar path from a Feff calculation on the bulk, crystalline analog to your cluster.

METHODS

fpath

Return a Demeter::FPath object representing the sum of the bins of the histogram extracted from the cluster.

plot

Make a plot of the the RDF.

BUGS AND LIMITATIONS

  • This currently only works for a monoatomic cluster. See rdf in SS.pm for species checks
  • Feff interaction is a bit unclear
  • Triangles and nearly colinear paths
