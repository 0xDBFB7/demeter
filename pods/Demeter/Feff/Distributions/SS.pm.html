NAME

Demeter::Feff::Distributions::SS - Histograms forsingle scattering paths

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

DESCRIPTION

This provides methods for generating histograms in path length for single scattering paths. It also provides a way to compute the triple scattering contribution of the sort:

  Abs ---> Scat. ---> Abs ---> Scat. ---> Abs

This is the path that rattles between the absorber and a neighbor. In general, this is only observable for the nearest neighbor.

Given a radial ranges for the scattering shell, this will dig through a configurational distribution and construct a histogram to describe the radial istribution function of that scattering atom. It then makes a Demeter::SSPath at each histogram bin, then sums them into a Demeter::FPath to make a single path-like object describing the single scattering (or rattle) contribution from that histogram.

ATTRIBUTES

file (string)

The path to and name of the HISTORY file. Setting this will trigger reading of the file and construction of a histogram using the values of the other attributes.

nsteps (integer)

When the HISTORY file is first read, it will be parsed to obtain the number of time steps contained in the file. This number will be stored in this attribute.

rmin and rmax (numbers)

The lower and upper bounds of the radial distribution function to extract from the cluster. These are set to values that include a single coordination shell when constructing input for an EXAFS fit. However, for constructing a plot of the RDF, it may be helpful to set these to cover a larger range of distances.

bin (number)

The width of the histogram bin to be extracted from the RDF.

sp (number)

This is set to the Demeter::ScatteringPath object used to construct the bins of the histogram. A good choice would be the similar path from a Feff calculation on the bulk, crystalline analog to your cluster.

rattle (boolean)

If true, the rattle contribution will be computed. If false, the single scattering contribution will be computed.

METHODS

fpath

Return a Demeter::FPath object representing the sum of the bins of the histogram extracted from the cluster.

plot

Make a plot of the the RDF histogram.
