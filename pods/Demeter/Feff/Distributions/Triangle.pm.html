NAME

Demeter::Feff::Distributions::Triangle - Histograms for short triangular paths

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

DESCRIPTION

This provides methods for generating two-dimensional histograms in path length and scattering angle for nearly short triangular arrangements of three atoms, like so:

   Absorber ---> Scatterer ---> Scatterer --+
     ^                                      |
     |                                      |
     +--------------------------------------+

Given two radial ranges for the nearer and more distant scatterers and bin sizes for

ATTRIBUTES

file (string)

The path to and name of the HISTORY file. Setting this will trigger reading of the file and construction of a histogram using the values of the other attributes.

nsteps (integer)

When the HISTORY file is first read, it will be parsed to obtain the number of time steps contained in the file. This number will be stored in this attribute.

r1 and r2; r3 and r4 (numbers)

The lower and upper bounds of the radial distribution function for the near and distant scatterer.

rbin (number)

The width of the histogram bin to be extracted from the RDF.

betabin (number)

The forward scattering angular range of the histogram bin to be extracted from the RDF.

METHODS

fpath

Return a Demeter::FPath object representing the sum of the bins of the histogram extracted from the cluster.

plot

Make a plot of the the RDF.
