NAME

Demeter::Feff::Distributions:: - Make historams from arbitrary clusters of atoms

VERSION

This documentation refers to Demeter version 0.9.14.

SYNOPSIS

DESCRIPTION

This provides support for importing data from clusters computed by molecular dynamics and making histograms for different types of paths.

This takes methods for parsing files of atomic configurations from its roles:

  Demeter::Feff::MD::Null
  Demeter::Feff::MD::DL_POLY
  Demeter::Feff::MD::VASP

This takes methods for making distributions functions and chi(k) (in the form of a Demeter::FPath object from its roles:

  Demeter::Feff::Distributions::SS
  Demeter::Feff::Distributions::NCL
  Demeter::Feff::Distributions::Thru

ATTRIBUTES

file (string)

The path to and name of the HISTORY file. Setting this will trigger reading of the file and construction of a histogram using the values of the other attributes.

nsteps (integer)

When the HISTORY file is first read, it will be parsed to obtain the number of time steps contained in the file. This number will be stored in this attribute.

rmin and rmax (numbers)

The lower and upper bounds of the radial distribution function to extract from the cluster. These are set to values that include a single coordination shell when constructing input for an EXAFS fit. However, for constructing a plot of the RDF, it may be helpful to set these to cover a larger range of distances.

bin (number)

The width of the histogram bin to be extracted from the RDF.

sp (number)

This is set to the Demeter::ScatteringPath object used to construct the bins of the histogram. A good choice would be the similar path from a Feff calculation on the bulk, crystalline analog to your cluster.

METHODS

fpath

Return a Demeter::FPath object representing the sum of the bins of the histogram extracted from the cluster.

plot

Make a plot of the the RDF.

CONFIGURATION

See Demeter::Config for a description of the configuration system. Many attributes of a Data object can be configured via the configuration system. See, among others, the bkg, fft, bft, and fit configuration groups.

BUGS AND LIMITATIONS

  • This currently only works for a monoatomic cluster.
  • Feff interaction is a bit unclear
  • Triangles and nearly colinear paths
