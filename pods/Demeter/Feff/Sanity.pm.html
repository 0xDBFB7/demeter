NAME

Demeter::Feff::Sanity - Sanity checks for feff.inp files

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

    $feff -> Demeter::Feff -> new(file=>"feff.inp");

DESCRIPTION

This module contains all the sanity checks made on a feff.inp file. This file forms part of the base of the Demeter::Feff class and serves no independent function. That is, using this module directly in a program does nothing useful -- it is purely a utility module for the Feff object.

The user should never need to call the methods explicitly since they are called automatically whenever a feff.inp file is imported. However they are documented here so that the scope of such checks made is clearly understood.

METHODS

The following sanity checks are made on the imported feff.inp file.

  • Check that all potential indeces are used in the atom list.

  • Check that no atoms have undefined potential indeces.

  • Check there is one and only one central atom, i.e. an atom with potential index 0.

  • Resequence the potential indeces if other tests pass but the indeces are not sequential.

  • Warn about rmax outside the cluster.

  • Warn if a potential index is greater than 7.

DIAGNOSTICS

I think these are all self-explanatory.

You have defined an atom with potential index greater than 7
You have not designated a central atom. No site in the atoms list is of potential #0.
You have defined more than one central atom by using potential #0 more than once in the atoms list.
You have used ipot #$i but not defined it as a potential.
You have defined ipot #$i but not used it in the atoms list.
You have specified a value of rmax that is outside the cluster.
