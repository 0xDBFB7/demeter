NAME

Demeter::Path::Process - Plotting and Fourier transform methods for Feff paths

VERSION

This documentation refers to Demeter version 0.9.14.

DESCRIPTION

This role of Demeter::Path contains methods for performing Fourier transforms. This is also used as a role by the VPath object.

METHODS

In practice, it should rarely be necessary to explicitly call these methods. When you plot paths or peform other chores, Demeter will notice whether the Fourier transform is up-to-date and will call these methods as needed.

fft

Perform a forward Fourier transform by generating and disposing of the appropriate sequence of Ifeffit or Larch commands.

  $path_object -> fft;
bft

Perform a backward Fourier transform by generating and disposing of the appropriate sequence of Ifeffit or Larch commands.

  $path_object -> bft;
