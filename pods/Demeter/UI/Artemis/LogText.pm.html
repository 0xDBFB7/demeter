NAME

Demeter::UI::Artemis::LogText - Add some color to a logfile

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

This is used by Demeter::UI::Artemis::Log and Demeter::UI::Artemis::History to colorize the display of log files from fits.

METHOD

This module provides the make_text method, which performs the colorization by lexically analyzing the text of the log file. It then places the colorized text in the specified Wx::TextCtrl.

    Demeter::UI::Artemis::LogText->make_text($textcrtl, $fit);

The first argument is a reference to the Wx::TextCrtl widget which will hold the text of the log file. The second argument is the Demeter::Fit object of the fit for which the log is being displayed.
