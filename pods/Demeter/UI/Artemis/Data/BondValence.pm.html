NAME

Demeter::UI::Artemis::Data::BondValence - Dialog to perform a simple bond valence sum calculation

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

This module provides a dialog for computing a bond valence sum using a set of marked paths and table of bond valence parameters maintained by I. David Brown of McMaster University.

DEPENDENCIES

Demeter's dependencies are in the Build.PL file.

BUGS AND LIMITATIONS

Please report problems to the Ifeffit Mailing List (http://cars9.uchicago.edu/mailman/listinfo/ifeffit/)

Patches are welcome.
