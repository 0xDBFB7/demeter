NAME

Demeter::UI::Hephaestus::Common - Common functions used in Hephaestus

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

This module contains functions used by many parts of Hephaestus.

  use Demeter::UI::Hephaestus::Common qw(e2l);

DESCRIPTION

Several common functions are conatined in this moduel for use throughout Hephaestus.

e2l

Convert between energy and wavelength.

  $l = e2l($e);
   #  or
  $e = e2l($l);
hversion

Return a string giving the Hephaestus version number.

hcopyright

Return a string giving the Hephaestus copyright statement.

hdescription

Return a string giving a description of Hephaestus' operating environment, including version numbers for perl, WxWidgets, and WxPerl.

enable_element

Enable/disable a specific element on a periodic table given a code reference that returns a true/false value.

   enable_element($pt, $el, $coderef)

The $coderef is called with $el as its argument. el is the first-letter capitalized element symbol. $pt is a pointer to an instance of a periodic table widget. The state of the $el button on $<pt> will be set.
