NAME

Demeter::UI::Wx::ColourDatabase - An objective interface to Wx::ColourDatabase

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

This is a pure perl implementation of Wx::ColourDatabase which implements those colors named by X11's rgb.txt file and those named by gnuplot.

  use Demeter::UI::Wx::ColourDatabase;
  my $cdb = Demeter::UI::Wx::ColourDatabase->new;
  my $wx_color_object = $cdb -> Find("yellowgreen");

The semantics are very similar to those documented for Wx::ColourDatabase, except that there is a constructor and object, rather than just class methods.

DESCRIPTION

This is an object oriented interface to the functionality provided by Wx::ColourDatabase and extended to include the named colors from X11's rgb.txt file and from gnuplot. The purpose of this object is to simplify the interaction with Wx colored windows while using named colors. The Add, Find, and FindName methods from Wx::ColourDatabase are all implemented.

Most of this was swiped from a post to the wxperl-users mailing list, http://www.nntp.perl.org/group/perl.wxperl.users/2007/06/msg3756.html by Mark Dootson on June 28, 2007.

Bruce added the interface to X11 and gnuplot colors.

DEPENDENCIES

Demeter's dependencies are in the Build.PL file. This requires Wx (of course) and File::Spec as well as files from a properly installed Demeter distribution.
