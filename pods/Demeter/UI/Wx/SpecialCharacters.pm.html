NAME

Demeter::UI::Wx::SpecialCharacters - A library of special characters for use in Wx labels

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

This provides a library of special characters for use in labels on Wx widgets used in Artemis and Athena.

  use Demeter::UI::Wx::SpecialCharacters qw(:all);
  my $button = Wx::Button($parent, -1, "Plot $CHI(k)");

DESCRIPTION

This collects all the various special characters needed in Artemis and other Wx-based GUIs into one convenient location. Please note that this exports variables into your module's namespace. This is a against good practice, but is a huge convenience and I am telling you about it!

The one exported function is emph, which encloses a string in some kind of quotation marks.

The greek letters exported are

$MU

lower case mu

$CHI

lower case chi

$EPSILON

lower case epsilon

$DELTA

upper case delta

$SIGMA

lower case sigma

$SIGSQR

lower case sigma followed by a proper superscripted 2

$PHI

lower case phi

$ETA

lower case eta

The superscript charaters exported are

$TWO

Superscript 2

$THR

Superscript 3

The other characters exported are

$COPYRIGHT

copyright symbol

$LABELS

left guillemet

$RAQUO

right guillemet

The sets exported are

:all

As the name implies, all defined characters plus the emph function

:greek

Just the greek letters

:super

Just the superscript characters

See http://en.wikipedia.org/wiki/List_of_Unicode_characters for the character codes.

To convert a hex number in decimal:

  perl -e 'printf "%d\n", 0x00BB'
