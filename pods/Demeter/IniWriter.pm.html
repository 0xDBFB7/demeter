NAME

Demeter::IniWriter -- Ini file writer for Demeter

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

This inherits from Config::INI::Writer, changing the write_handle method to allow for saving items to the mru lists that contain UTF-8 path and filenames without triggering the "wide character" warning.

ACKNOWELDGEMENT

Config::INI::Writer was written by Ricardo Signes <rjbs@cpan.org>. This is just a thin wrapper around that module.
