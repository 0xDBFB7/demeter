NAME

Demeter::Files - File import tests

VERSION

This documentation refers to Demeter version 0.9.26.

DESCRIPTION

This role contains several methods for identifying files common to the Demeter, Feff, and Ifeffit/Larch universes.

METHODS

is_atoms

Return true if a file is recognized as an atoms input file. This uses simple semantics based on the file contents.

  my $yesno = Demeter->is_atoms($file, $verbose);
is_cif

Return true if a file is recognized as a CIF file. This is a pretty dumb method -- it returns true if the file extension is .cif.

  my $yesno = Demeter->is_cif($file, $verbose);
is_feff

Return true if a file is recognized as a Feff input file. This uses simple semantics based on the file contents.

  my $yesno = Demeter->is_feff($file, $verbose);
is_data

Return true is Ifeffit/Larch recognizes this file as a data file.

  my $yesno = Demeter->is_data($file, $verbose);
is_prj

Return true if this is a conventional Athena project file.

  my $yesno = Demeter->is_prj($file, $verbose);
is_json

Return true if this is a JSON-style Athena project file.

  my $yesno = Demeter->is_json($file, $verbose);
is_zipproj

Return true if this is a zip file which can be recognized as a save file for some aspect of Artemis.

  my $yesno = Demeter->is_zipproj($file, $verbose, $type);

$type can be one of:

any

Returns true if the file is any kind of recognized save file in a zip format.

guess

Returns true if the file is any kind of recognized save file in a zip format. The return value tells you what kind. -1 means an Artemis project file., -2 means a Demeter fit serialization file, and -3 means an old-style Artemis project file.

fpj

Returns true only if the file is an Artemis project file.

dpj

Returns true only if the file is a Demeter fit serialization file.

apj

Returns true only if the file is an old-style Artemis project file.

is_xdi

Return true if the file is recognizably an XDI file.

  my $yesno = Demeter->is_xdi($file, $verbose);
