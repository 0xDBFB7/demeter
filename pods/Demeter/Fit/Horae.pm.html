NAME

Demeter::Fit::Horae - Convert an old-style Artemis project file into a Demeter fit serialization

VERSION

This documentation refers to Demeter version 0.9.26.

DESCRIPTION

Convert an old-style Artemis project file file into the equivalent Demeter fit serialization. This is a role of the Fit object.

   my $fit = Demeter::Fit->new();
   $fit -> apj2dpj($apjfile, $dpjfile, $rjournal);

This is currently in use by Artemis to import projects from the older version.

METHODS

There is only one outwardly visible method, apj2dpj. It takes three arguments:

  1. The filename of the old-style .apj file.

  2. The filename of the fit serialization, i.e. .dpj file. This is the output of this method. In Artemis, this is a temporary file in the stash folder which is unlinked after being imported into Artemis.

  3. A reference to a scalar which will be filled with the contents of the journal from the .apj file. In Artemis, this is inserted as text into the Journal buffer.

CONFIGURATION

There are no configuration options for this role.

See Demeter::Config for a description of Demeter's configuration system.

DEPENDENCIES

Demeter's dependencies are in the Build.PL file.
