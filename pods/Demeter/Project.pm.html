NAME

Demeter::Project - Project file management

VERSION

This documentation refers to Demeter version 0.9.26.

DESCRIPTION

This module contains various utilities involved in project file management for the demeter system. It provides tools for finding and making the various temporary directories Demeter uses as workspace and a method for packaging a workspace up into a zip file.

METHODS

There are no user methods. This module contains tools used by other modules.

DIAGNOSTICS

None.

CONFIGURATION AND ENVIRONMENT

See Demeter::Config for a description of the configuration system.

DEPENDENCIES

Demeter's dependencies are in the Build.PL file.

BUGS AND LIMITATIONS
