NAME

Demeter::Plugin::Beamlines::X11A - beamline recognition plugin for files from pre-XDAC X11A

VERSION

This documentation refers to Demeter version 0.9.22.

SYNOPSIS

This plugin recognizes files measured using EDC, the long-lost data acquisition program at NSLS X11A.

For details about Demeter beamline recognition plugins, see Demeter::Data::Beamlines.

For information about the XAS Data Interchange format, see https://github.com/XraySpectroscopy/XAS-Data-Interchange
