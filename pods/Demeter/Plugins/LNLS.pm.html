NAME

Demeter::Plugin::LNLS - Import data from the XAS beamlines at LNLS

VERSION

This documentation refers to Demeter version 0.9.22.

SYNOPSIS

This plugin makes an LNLS file readable by Athena by stripping the first two columns from the data file.

Methods

is

Recognize the LNLS file by the first line, which contains the words "Data" and "Hora" and by subsequent lines, which have dates and times rather than nice, sensible numbers in the first two columns.

fix

Strip the first two columns from the LNLS data file.
