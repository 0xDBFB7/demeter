NAME

Demeter::Plugin::B18 - Diamond B18 filetype plugin

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

This plugin rationalizes a file from B18's 36-element detector to make importing it into Athena less painful.

Methods

is

The is method is used to identify the file type, typically by some information contained within the file. In the case of ESRF B18 data, the file is recognized by the strings "Diamond" and "B18-CORE XAS" in the first two line.

fix

Remove tabs and leading spaces from the data table, filter out every other line to make the file of an ifeffit-friendly length, and make a good suggestion for column selection.
