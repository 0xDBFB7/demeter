NAME

Demeter::Plugin::X10C - NSLS X10C filetype plugin

VERSION

This documentation refers to Demeter version 0.9.17.

SYNOPSIS

This plugin directly imports files from NSLS beamline X10C

Methods

is

The is method is used to identify the file type, typically by some information contained within the file. In the case of an NSLS X10C data, the file is recognized by the string "EXAFS" on the first line and by the string "DATA START" several lines later.

fix

For an NSLS X10C file, the null characters are stripped from the header, the header lines are commented out with hash characters, and the situation of the fifth data column not being preceeded by white space is corrected.
