NAME

Demeter::Plugins::SpecFileLongLine - Deal with SPEC files with a very long column label line

VERSION

This documentation refers to Demeter version 0.9.20.

SYNOPSIS

This plugin cleans up a SPEC file that has a column label line -- the one with #L -- that is too long for the hard-wired string length in Ifeffit.

Methods

is

The is method is used to identify the file type by noticing a #L line that is longer than 254 characters.

fix

This cleans up the file by simply copying every line except the troublesome one.
