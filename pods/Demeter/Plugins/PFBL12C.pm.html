NAME

Demeter::Plugin::PFBL12C - filetype plugin for Photon Factory and SPring8

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

This plugin converts data recorded as a function of mono angle to data as a function of energy.

METHODS

is

This file is identified by the string "KEK-PF" or "SPring-8" followed by the beamline number in the first line of the file.

fix

Convert the wavelength array to energy using the formula

   data.energy = 2 * pi * hbarc / 2D * sin(data.angle)

where hbarc=1973.27053324 is the the value in eV*angstrom units and D is the Si(111) plane spacing.
