NAME

Demeter::Plugin::SLRIBL4 - filetype plugin for SLRI BL4 dispersive XAS data

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

This plugin directly imports files from SLRI beamline 4

Methods

is

Recognize data from SLRI BL4 by ....

fix

Using the calibration parameters already found using Athena's dispesive XAS tool (and stored on disk in an initialization file), convert DXAS data from SLRI BL4 to an energy axis and write the mu(E) result to a temporaray file in the stash folder.

BUGS AND LIMITATIONS

  • Need to make sure athena.dxas exists and has useful values.
