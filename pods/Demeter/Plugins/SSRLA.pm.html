NAME

Demeter::Plugin::SSRLA - SSRL XAFS Data Collector 1.3 ASCII filetype plugin

SYNOPSIS

This plugin directly reads the files written by the SSRL XAFS Data Collector.

SSRL files

This plugin comments out the header lines, constructs a column label line out of the Data: section, moves the first column (real time clock) to the third column, and swaps the requested and acheived energy columns.
