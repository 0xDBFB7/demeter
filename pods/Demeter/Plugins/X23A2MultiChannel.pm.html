NAME

Demeter::Plugin::X23A2MultiChannel - filetype plugin for X23A2 multi-channel data files

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

This plugin converts data from X23A2 multi-channel ion chambers to an Athena project file.

METHODS

is

The multichannel file is recognized by "XDAC" in the first header line and

  I0   I02  I03  I04  It1  It2  It3  It4

in that order as column headers.

fix

The column data file is converted into an Athena project file using the Demeter::Data::MultiChannel object.

BUGS AND SHORTCOMINGS

  • Need to prompt for label columns

  • Tie all four channels

  • Prompt whether to import reference, whether to use all 4 It channels in the numerator, tie reference to all four groups

  • Need a persistance file for label columns and reference
