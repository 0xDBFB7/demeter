NAME

Demeter::Plugin::CMC - filetype plugin for files from APS Sector 9

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

This plugin strips the many columns not normally needed from a file from CMC APS Sector 9 in an effort to interact with Ifeffit more efficiently and avoid some of the pitfalls of the CMC file format.

This plugin is used with the SPEC file generated using the smaller, upstream station and is not necessary for data files obtained from the newer control system using the software orginally fro 20BM.

Methods

is

Recognize the Sector 9 BM file by the fourth line, which starts with the Spec #C token and identifies the user as "bmexafs".

fix

Strip out all columns except for energy, I0, I1, I2, Lytle, and mca* and write them to a file in the stash directory. Remove dark current from the i* channels. Also fix any instances of NaN among the data (probably only in the ill-conceived logi0i1 column, but...) and strip out any blank lines.
