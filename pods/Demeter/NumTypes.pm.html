NAME

Demeter::NumTypes - Numerical type constraints

VERSION

This documentation refers to Demeter version 0.9.23.

DESCRIPTION

This module implements numerical type constraints for Moose using MooseX::Types.

CONFIGURATION AND ENVIRONMENT

See Demeter::Config for a description of the configuration system.
