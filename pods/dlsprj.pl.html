NAME

lsprj - List contents of an Athena project file

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

  lsprj my.prj

DESCRIPTION

This program writes out a list of the labels for each group in an Athena project file. This, then, is a quick way of peering inside a project file without actually opening it.

The data from the project file can also be plotted by supplying the optinal --plot command line switch. This is a quick-and-dirty plotting tool intended for taking a peek inside a project file rather than providing fine control over the details of the plot.

A short usage message will be printed to the screen if you leave off the filename at the command line.

COMMAND LINE SWITCHES

--plot or -p

Optionally plot the data from the project file in the specified plotting space. The plotting options are

e

plot mu(E)

n

plot normalized mu(E)

d

plot derivative mu(E)

k

plot k-weighted chi(k)

r

plot the magnitude of chi(R) Fourier tranformed with k^2 weighting

q

plot the real part of chi(q)

Values for data processing parameters are taken from the Athena project file. This is intended for quick-n-dirty display, so very few plotting options are available.

Note that for large project files, the plot will be time consuming and cluttered. This option is most useful for smaller project files.

--stack

Data plotted as normalized mu(E), chi(k), chi(R), or chi(q) are stacked, with each subsequent trace being shifted downward by 0.2. The amount of the shift can be specified with this switch.

-kw or -kweight

Override the default k-weight (which is 2).

--xanes

When plotting mu(E) or normalized mu(E) this flag restricts the plot to 50 volts below and 120 volts above the edge.

--edge

When plotting mu(E) or normalized mu(E) this flag restricts the plot to 20 volts below and 60 volts above the edge.

--emin and --emax

To fine tune the plotting ranges in energy beyond the setting associated with --xanes or --edge, you can use these two switches, which take energy vlues relative to the edge.

--params

A comma separated list of data processing parameters to include in the list of groups. This is a bit awkward to use in that you must specify the parameters using their internal representations as given explained in the documentation for Demeter::Data. For instance, this will show the values for the background removal R_bkg and the forward Fourier transform kmin value:

  lsprj my.prj --params=bkg_rbkg,fft_kmin,
   ==prints==>
    #       record     bkg_rbkg   fft_kmin
    # ----------------------------------------------------
        1 : HgO        1.0        2
        2 : HgS black  1.0        2
        3 : HgS red    1.0        2

ENVIRONMENT VARIABLE

Command line switches can be specified via an environment variable called LSPRJ. The contents of that variable will be parsed before the specified command line variables. That is, the flags specified on the command line over-ride the contents of the LSPRJ environment variable. For example:

   export LSPRJ="--plot=e --edge -g"
   lsprj my.prj 7,12,17

This does the same as

   lsprj my.prj --plot=e --edge -g 7,12,17
