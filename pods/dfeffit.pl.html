NAME

dfeffit - Read and use a feffit.inp file

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

Process a feffit.inp file with Demeter.

  dfeffit [-r -i -p -q] inputfile savefile

If no save file is given on the command line, the script will be echoed to the screen.

DESCRIPTION

This script is a wrapper around Demeter::Fit::Feffit and can be used to run the fit encoded in a feffit input file or to convert the input file either to an Ifeffit script or to a perl script using Demeter.

COMMAND LINE SWITCHES

-r

Run a fit using the information in the feffit.inp file. Specifying this flag will override either of the -i or -p flags when (for some reason) more than one is specified on the command line.

If the fit is run, a log file and a project file will be written and the interview method will be called. See Demeter::UI::Screen::Interview.

-i

Generate an ifeffit script which is equvalent to the feffit.inp file.

-p

Generate a perl script which uses Demeter and is equvalent to the feffit.inp file. This is the default behavior.

-q

Suppress the echoing of the script to the screen. This is useful with the -r flag.
