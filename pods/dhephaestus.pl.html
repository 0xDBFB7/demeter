NAME

hephaestus - A souped-up periodic table for the X-ray absorption spectroscopist

VERSION

This documentation refers to Demeter version 0.9.26.

DESCRIPTION

Hephaestus is a graphical interface to tables of X-ray absorption coefficients and elemental data. The utilities contained in Hephaestus serve a wide variety of useful functions as you prepare for and perform an XAS experiment.
