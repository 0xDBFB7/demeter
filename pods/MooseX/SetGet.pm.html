NAME

MooseX::SetGet - Moose Role to add a quick set and get methods

VERSION

This documentation refers to Demeter version 0.9.26.

SYNOPSIS

This swipes the set method from MooseX::MutatorAttributes and adds a get method. Like the set from MooseX::MutatorAttributes, this one returns self. The get method will return a scalar or an array.

    with qw{MooseX::MutatorAttributes};
    $obj->set( attr1 => $value1, attr2 => $value2 )->method_that_uses_attr;
    $value  = $obj->get('attr1');
    @values = $obj->get(qw(attr1 attr2));

METHODS

set

    $self->set( HASH );

This takes a hash, keys are expected to be attributes, if they are not then we Carp::croak. If a key is an acceptable attribute then we attempt to set with $value.

get

    $value  = $self->get("attribute");
    @values = $self->get(@attributes);

This takes a list of one or more attributes and returns either a scalar or an array containing the values associated with the attributes.

ACKNOWLEDGEMENTS
