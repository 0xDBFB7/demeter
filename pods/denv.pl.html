NAME

denv - Display Demeter's operating environment

VERSION

This documentation refers to Demeter version 0.9.26.

DESCRIPTION

This simple tool calls the module_environment method and prints it's output to the screen. It's sole purpose in life is as a diagnostic tool for understanding the environment in which Demeter is running.

OPTIONS

--wx

Append information about Wx and WxPerl to the report.
